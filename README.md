# 🎬 CineMatch AI: Personalized Movie Recommendation Engine

CineMatch AI is a Machine Learning-powered movie recommendation system designed to help users discover movies similar to their favorites. The system uses a content-based filtering approach and cosine similarity algorithms to analyze movie characteristics and generate intelligent recommendations.

The recommendation engine processes multiple movie attributes such as genres, keywords, cast members, directors, and movie overviews to understand content relationships between films. By converting textual movie information into numerical vectors and measuring similarity scores, CineMatch AI provides personalized recommendations with high relevance.

To improve user experience and create a visually engaging application, TMDB API integration has been used to dynamically fetch movie posters. The project also features an interactive Streamlit interface with a clean movie-browsing experience.
